---
title: GET hordes
---

- Returns an object array.  
- It provides all the information about hordes.

## Parameters

Parameter   | Type    | Description
----      | ----    | ----
stat      | String  | Filter by stat name.
game      | Integer   | Filter by game edition.


### Example request

```
GET {{ site.api_url }}/v1/hordes
```
