---
title: GET trainings/:id
---

Returns a specific training.

## Parameters

Parameter       | Type          		| Description
---- | ---- | ---- 
id _required_   | String			   	| Refers to the training ID


### Example request 

```
GET https://api.evscalculator.com/v1/trainings/bglRdqG4no
```

{
